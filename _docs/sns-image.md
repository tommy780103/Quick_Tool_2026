# SNS画像作成ツール（sns-image）

## 概要
「画像加工」カテゴリに、SNS投稿用の画像を作成するツールを追加した。
指定したSNSサイズのキャンバスを最大10フレームに分割し、各フレームに写真を配置（コラージュ）、
さらにテキストを自由に載せて1枚の画像として書き出す。Fabric.js（既存読み込み）を利用。

## 機能
- **サイズプリセット**：YouTubeサムネ(1280×720) / TikTok・リール・ショート(1080×1920) / Instagram 正方形(1080×1080)・縦長(1080×1350)・ストーリー(1080×1920) / X横長(1600×900) / カスタム(px指定)。
- **写真分割（コラージュ型）**：フレーム数 1〜10、レイアウト（グリッド／横並び／縦並び）、フレーム間余白、背景色を指定。各フレームをクリックで写真を配置（cover表示）。画像は選択してドラッグ／拡大で枠内の見え方を調整、ダブルクリックで差し替え。
- **テキスト（自由編集）**：テキスト追加、フォント・サイズ・色・太字・フチ取り。ドラッグで移動、ダブルクリックで文字編集、複数追加可。Deleteキー／削除ボタンで除去。
- **書き出し**：PNG / JPG。プリセットの実寸（例 1280×720）で出力。

## 設計メモ（Fabric.js）
- 内部キャンバスは実寸。表示は `setZoom` + `setWidth/Height` で縮小（既存 `image-edit.js` と同方式）。書き出しは一旦 `setZoom(1)`＋実寸に戻して `toDataURL`、その後復元。
- フレームは `computeCells()` がセル矩形 `{x,y,w,h}` を算出。空フレームは破線の `fabric.Rect`、写真は `fabric.Image` に `clipPath`（`absolutePositioned` のセル矩形）を設定して cover 配置。
- レイアウトやフレーム数を変えても写真（dataURL）とテキストは保持。`reorderLayers()` で「枠→画像→テキスト」の重なり順を維持。
- パネルは初期非表示のため、`#sns-image` 表示時に `requestAnimationFrame` 経由で遅延初期化（サイズ確定後にズーム適用）。
