# カラーピッカー：メインカラー & 補色の追加

## 概要
カラーピッカーツールに「メインカラー（現在の色）と補色」を並べて表示するセクションを追加した。

## 補色の定義
カラーホイール上で正反対の色 = **色相（Hue）+180°、彩度・明度は保持**。
（RGB反転 `255-x` の「反転色」とは別物。デザイン用語の「補色」に合わせた）

## 追加・変更したファイル
| ファイル | 変更内容 |
| --- | --- |
| `index.html` | カラーピッカーパネルのスライダー直後に「メインカラー & 補色」セクション（`.cp-harmony`）を追加 |
| `js/tools/color-picker.js` | `hslToRgb` / `hue2rgb` / `complementaryRgb` を追加。`updateAll` でメイン＋補色を描画。メイン＝クリックでHEXコピー、補色＝クリックでその色に切替 |
| `css/tools.css` | 末尾に `.cp-harmony*` スタイルを追加 |

## 動作
- 色を変更（ピッカー／スライダー／プリセット／保存色／スポイト／画像）すると、`updateAll()` 経由でメイン色と補色が自動更新される。
- **メイン**スウォッチをクリック → そのHEXをコピー。
- **補色**スウォッチをクリック → その色を現在の色として適用（ピッカー全体が補色に切り替わる）。

## 検証（Nodeで確認済み）
| 入力 | 補色 | 期待 |
| --- | --- | --- |
| #FF0000（赤） | #00FFFF | シアン ✓ |
| #00FF00（緑） | #FF00FF | マゼンタ ✓ |
| #0000FF（青） | #FFFF00 | 黄 ✓ |
| #808080（グレー） | #808080 | 無彩色は不変 ✓ |
| 補色の補色（#FF0000） | #FF0000 | 元の色に戻る（ロスレス）✓ |

- `node --check` 構文OK。新規ID 6点（cp-main/comp-item/swatch/hex）はHTML/JSで一致。
