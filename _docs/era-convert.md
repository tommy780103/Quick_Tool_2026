# 西暦和暦変換ツール（era-convert）

## 概要
「便利ツール」カテゴリに、西暦と和暦（元号）を相互変換するツールを追加した。
明治・大正・昭和・平成・令和の5元号に対応。完全にクライアントサイドで動作する。

## 追加・変更したファイル
| ファイル | 変更内容 |
| --- | --- |
| `index.html` | サイドバー項目追加（便利ツール 5→6）／ホームカード追加／ツールパネル `#tool-era-convert` 追加／`era-convert.js` 読み込み追加 |
| `js/tools/era-convert.js` | 新規。変換ロジック・早見表・イベント処理 |
| `css/tools.css` | 末尾に `.ec-*` スタイルを追加 |

## 機能
1. **西暦 → 和暦**
   - 年（必須）／月・日（任意）を入力するとリアルタイム変換。
   - 月日まで入力すると改元日を厳密に判定（例: 2019/4/30 → 平成31年、2019/5/1 → 令和元年）。
   - 年のみ入力で、その年が改元年の場合は両元号を期間付きで表示（例: 2019 → 平成31年（〜4月30日）／令和元年（5月1日〜））。
   - 「今日の日付」ボタンで現在日付をセット。
2. **和暦 → 西暦**
   - 元号セレクト＋年（元年=1）＋月・日（任意）。
   - 閉じた元号で最終年を超える入力には警告を表示（例: 平成32年 → 注記）。
3. **元号早見表** — 各元号の期間・元年・最終年を一覧表示。
4. 各結果に「コピー」ボタン（`navigator.clipboard`）。

## 元号データ（改元日・グレゴリオ暦）
| 元号 | 開始 | 最終年 |
| --- | --- | --- |
| 令和 | 2019-05-01 | （現在） |
| 平成 | 1989-01-08 | 平成31年（2019） |
| 昭和 | 1926-12-25 | 昭和64年（1989） |
| 大正 | 1912-07-30 | 大正15年（1926） |
| 明治 | 1868-01-01 ※元年=1868年を通年扱い | 明治45年（1912） |

- 1868年より前は「対応範囲外」として扱う。
- 明治元年は慣例に従い1868年を通年で明治元年とする（改元の詔書による遡及適用）。

## 実装メモ
- 日付比較は `年*10000 + 月*100 + 日` の数値キーで行う。
- 元号の終了境界は「次の元号の改元日の前日」を実日付（`Date.setDate(-1)`）で算出。
  当初は「開始キー−1」で求めていたが、平成終了が `20190500`（5月0日）となり
  早見表記の「〜○月○日」で日が0になる不具合があったため修正済み。
- 変換ロジックの境界（4/30↔5/1、1/7↔1/8、12/24↔12/25、7/29↔7/30）はNodeで検証済み。
